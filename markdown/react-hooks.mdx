## 1. Understanding Flexbox Layout

### What is Flexbox?

Flexbox is a CSS layout model that allows you to design complex layouts with ease. It enables dynamic resizing and alignment of elements within a container, making it perfect for responsive design.

### Flexbox Example: Centering Items

```css
.container {
  display: flex;
  justify-content: center; /* Center items horizontally */
  align-items: center; /* Center items vertically */
  height: 100vh; /* Full viewport height */
}

.item {
  padding: 10px;
  background-color: #4caf50;
  color: white;
}
```
